# Personal Profile Website

This repository contains my **personal profile and background website** built using **Vue.js**.  
It showcases my personal details, background information, and serves as a simple online presence.


## About the Project

The goal of this project is to:
- Present my personal and professional background
- Practice building a structured frontend application with Vue.js
- Create a reusable foundation for a personal or portfolio website

## Features

- Personal details and background information
- Clean and simple layout
- Responsive design
- Component-based structure using Vue.js

## Tech Stack

- **Vue.js**
- JavaScript
- HTML / CSS
- **Vuetify**
